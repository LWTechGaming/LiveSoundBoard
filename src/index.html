About Live Sound Board

Live Sound Board is an application that I originally built for Lovisa Gymnasium's Christmas play. We needed a custom-made sound board, and any pre-made program wasn't going to cut it.

Thus, I set about making this program. Here's a list of programs and frameworks used to make it.

Frameworks
Electron - Desktop application framework
Materialize - User interface design
NPM modules
sound-board - Sound downloading and playing

Source code

This project is open source and the source code is available on GitHub.
